# Запуск проекта в Docker контейнере
```bash
- git clone https://github.com/madpenguinw/Goss_Practice_Frontend
- запустить docker
- . run-prod.sh
```
Или можно использовать локальный сервер
```bash
- npm install --global yarn
- yarn && yarn run dev
```
